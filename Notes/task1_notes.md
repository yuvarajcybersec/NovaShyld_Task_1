
# Task 1 – Foundations of Ethical Hacking

## 1. Ethical Hacking vs Illegal Hacking

Ethical hacking is the process of testing systems legally with permission to identify vulnerabilities. Illegal hacking involves unauthorized access to systems.

## 2. Scope & Permission

* Testing must be within defined scope
* Permission is mandatory
* Responsible disclosure is required

## 3. Pentesting Methodology

1. Reconnaissance – Gathering information
2. Scanning – Identifying open ports/services
3. Exploitation – Attempting to exploit vulnerabilities
4. Reporting – Documenting findings

## 4. Linux Basics

* pwd: shows current directory
* ls: lists files
* cd: change directory
* cp: copy files
* mv: move files
* rm: remove files
* chmod: change permissions
* apt: install packages

## 5. Networking Basics

* IP Address: unique identifier of device
* Ports: communication endpoints
* TCP/UDP: protocols
* DNS: domain to IP resolution
* HTTP vs HTTPS: secure vs non-secure communication

## 6. Basic Commands Used

* ip a
* ping
* traceroute
* ss
* nmap
