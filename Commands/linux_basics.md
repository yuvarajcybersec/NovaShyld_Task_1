# Linux Essentials for Ethical Hacking

## Navigation Commands
- pwd → Shows current directory
- ls → Lists files and folders
- cd → Changes directory

## File Handling
- mkdir → Creates directory
- touch → Creates file
- cp → Copies file
- mv → Moves or renames file
- rm → Deletes file

## Permissions
- chmod → Changes file permissions

Example:
chmod +x file → makes file executable

## Package Management
- apt → Used to install/update software

Example:
sudo apt update → Updates package list

## Importance in Ethical Hacking
Linux is widely used in cybersecurity because most tools run on Linux environments. Understanding basic commands is essential for performing penetration testing tasks.
